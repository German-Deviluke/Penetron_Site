Остальные Материалы

Скрепа финишная

Скрепа финишная — тонкослойная выравнивающая смесь для финишной отделки бетонных и кирпичных поверхностей.

Скрепа Реставрационная

Скрепа Реставрационная — ремонтный состав для восстановления геометрии бетонных элементов и устранения дефектов.

Скрепа Самонивелир

Скрепа Самонивелир — быстротвердеющий наливной пол для выравнивания бетонных оснований.

Скрепа Инъекционная

Скрепа Инъекционная — мелкодисперсный состав для инъекционного ремонта трещин и пустот в бетоне.

Скрепа M-500 Ремонтная

Скрепа M-500 Ремонтная — высокопрочный состав для локального ремонта и восстановления бетонных конструкций.

ПенеПурФом 65

ПенеПурФом 65 — однокомпонентный инъекционный состав для остановки активных течей и заполнения трещин.

Пенесплитсил

Пенесплитсил — герметизирующий состав для заполнения швов и трещин, устойчивый к агрессивным средам.

Скрепа Эластичная

Скрепа Эластичная — двухкомпонентная эластичная гидроизоляционная смесь для защиты и герметизации.

Пенепурфом

ПенеПурФом — однокомпонентный полиуретановый состав для быстрой герметизации активных протечек.

ПенеПурФом 1К

ПенеПурФом 1К — однокомпонентный полиуретановый состав для экстренного устранения протечек в конструкциях.

Ускоритель Пенепурадмикс

Ускоритель Пенепурадмикс — добавка для ускорения вспенивания и отверждения полиуретановых составов Пенепур.

Пенебанд

Пенебанд — эластичная гидроизоляционная лента для герметизации деформационных швов.
